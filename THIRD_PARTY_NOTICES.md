# Third-party software

Packaged OneCut releases contain Python, Pillow, FFmpeg, FFprobe, and the
libraries reported in the accompanying `ffmpeg-build.txt` release asset.

- Python is distributed under the Python Software Foundation License.
- Pillow is distributed under the HPND License. Its embedded Aileron font is
  distributed under the SIL Open Font License.
- FFmpeg and FFprobe are distributed under the license reported by that exact
  build. FFmpeg licensing and source information is available at
  <https://ffmpeg.org/legal.html> and <https://ffmpeg.org/download.html>.

OneCut itself is distributed under GPL-3.0. Each release records the complete
FFmpeg version and build configuration used to produce its executable.
